What is this?

Tag Locate is a demonstrator application for object-based indoor positioning, focusing mainly on NFC technology. However, it also supports QR codes.

How to locate

NFC: Put the device against a NFC tag that contains location information while the app is running.
QR: Press menu key, select "QR Locate". 'Barcode Scanner' must be installed for this to work.

Tag data format

Tag Locate supports Geo-URI's as well as iii-URI's containing a Geo-URI target element.
For QR-codes, use 'Geo-QR code', which is basically a Geo-URI.

ID resolution

In addition to explicit location URI's written on the tags, the unique ID of NFC tags may also be used for getting a location. In this case, the application uses a reference file that links those ID's with coordinates. QR codes may also be used as ID's, in which case their specific content will be no further evaluated. See the default reference file (/sdcard/taglocate/default.iir) for more information.

Resolution strategies

There are multiple strategies to get location information out of tags:

ID first: The tag will only be evaluated for an explicit location link, if it's ID cannot be found in the link file.
Link first: The ID will only be looked up if the tag doesn't contain an explicit location link.
ID only: Only the ID will be evaluated.
Link only: Only the payload data will be evaluated.

Files

The /taglocate/ folder on the SD card contains geometry files and link files. You can define your own files here. Note that the geometry files currently use a very simplistic OSM format, only supporting 'node' and 'way' elements.

File download from NFC

It is possible to store geometry and link files on NFC tags. Those files may be downloaded and applied. To create a NFC tag containing files, put the desired files into a ZIP archive and write the archive's byte array into a NDEF record using the MIME declaration 'application/zip'.

Reference new Tags

To reference new NFC tags or QR codes, select 'Add Tag' from the options menu. A crosshairs symbol will appear in the center of the map. Move the view so that the symbol marks the location you want to associate with the tag. Select 'Add Tag here'. Now you can choose between:

'Write NFC Tag': The location will be written to a NFC tag using a Geo-URI.
'Reference NFC ID': The tag's ID will be associated with a location within the local reference model (i.e. IIR file). If the ID has already been referenced, the reference will be updated with the new location.
'Reference QR Code': The QR code will be referenced in the local reference model just like a NFC ID. So make sure the QR's content is unique.
